## Installation & Setup
1. **Clone the repository:**
   ```bash
   git clone https://github.com/izaz-ahamed-mallick/MoviesApp.git
   cd MoviesApp
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```
   The app will be available at `http://localhost:5173`.
